# 강우량 예측 프로젝트

## 프로젝트 소개

본 프로젝트는 기상 데이터를 활용하여 미래 강우량을 예측하는 인공지능 프로젝트이다.

Random Forest와 LSTM 모델을 사용하여 강우량을 예측하고 성능을 비교하였다.

---

## 팀원

- 김진우
- 윤서연
- 오랑거

---

## 프로젝트 목표

- 과거 강우량 데이터를 이용하여 미래 강우량 예측
- 머신러닝과 딥러닝 모델 성능 비교
- 최적의 예측 모델 선정

---

## 사용 데이터

- 여수 2023 데이터
- 여수 2024 데이터
- 여수 2025 데이터
- 통영 2023 데이터
- 통영 2024 데이터
- 통영 2025 데이터

---

## 데이터 전처리

1. 날짜 데이터 변환
2. 결측치 제거
3. 30분 단위 리샘플링
4. 학습 데이터와 테스트 데이터 분리

---

## 사용 모델

### Random Forest

- 여러 개의 결정트리를 이용하는 앙상블 학습 모델
- 비선형 데이터를 효과적으로 학습 가능

### LSTM

- 시계열 데이터 분석에 특화된 딥러닝 모델
- 과거 패턴을 기억하여 미래 값을 예측

---

## 평가 지표

- MSE (Mean Squared Error)

---

## 실험 결과

| 모델 | MSE |
|-------|-------|
| Random Forest | 21.62 |
| LSTM | 65.38 |

---

## 결과 분석

**Random Forest 모델이 LSTM보다 낮은 MSE를 기록하였다.**

따라서 본 데이터셋에서 여수는 Random Forest가 더 우수한 예측 성능을 보였지만 통영은 LSTM이 더 우수한 예측 성능을 보여줬다.
