# 여수·통영 남해안 지역 강우 예측 모델 개발 프로젝트

## 프로젝트 소개

본 프로젝트는 여수·통영 남해안 지역의 기상 데이터를 활용하여 미래 강우량을 예측하고, 이를 기반으로 수위 감지 시스템의 정확도를 향상시키기 위해 수행되었다.

다양한 머신러닝 및 딥러닝 모델을 구현하고 비교하여 강우량 예측에 가장 적합한 모델을 선정하였다.

---

## 팀원

* 김진우

## 프로젝트 배경

최근 기후 변화와 기상 이변으로 인해 극단적인 강우 패턴이 증가하고 있다.

특히 여름철 집중호우는 홍수 및 침수 피해를 유발할 수 있으므로 정확한 강우량 예측이 중요하다.

본 프로젝트는 여수·통영 지역의 기상 데이터를 활용하여 강우량을 예측하고, 수위 감지 시스템에 적용 가능한 최적의 예측 모델을 탐색하는 것을 목표로 한다.

## 모델 설명

### Random Forest

여러 개의 결정트리를 활용하는 앙상블 머신러닝 모델로, 강우량 변화 패턴을 안정적으로 학습하여 예측을 수행한다.

### SVR

서포트 벡터 머신 기반 회귀 모델로, 비선형 데이터에 대한 예측이 가능하다.

### LightGBM

Gradient Boosting 기반 모델로 빠른 학습 속도와 우수한 예측 성능을 제공한다.

### LSTM

시계열 데이터의 시간적 특성을 학습하는 딥러닝 모델로, 과거 강우량 데이터를 기반으로 미래 강우량을 예측한다.

### AutoEncoder

데이터의 특징을 압축 및 복원하는 과정을 통해 패턴을 학습하며, 이상 탐지와 강수 발생 탐지에 효과적인 딥러닝 모델이다.

### Hybrid (1D-CNN + GRU)

CNN과 GRU를 결합한 모델로, 지역적 특징과 시계열 특징을 동시에 학습한다.

## 최종 모델 선정

### AutoEncoder

선정 이유

* LSTM 대비 전반적으로 우수한 예측 성능
* 강수 발생 시점을 안정적으로 탐지
* Precision과 Recall의 균형이 우수함
* 높은 F1-Score 기록
* 강수 피크 구간을 효과적으로 반영
* 지역이 달라져도 안정적인 성능 유지

따라서 본 프로젝트에서는 AutoEncoder를 최종 강우량 예측 모델로 선정하였다.
