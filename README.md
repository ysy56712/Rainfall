# 강우량 예측 프로젝트

## 프로젝트 소개

본 프로젝트는 기상 데이터를 활용하여 미래 강우량을 예측하는 인공지능 프로젝트이다.

팀원들이 각각 서로 다른 머신러닝 및 딥러닝 모델을 구현하여 강우량 예측을 수행하였으며, 모델별 성능을 비교하여 가장 우수한 모델을 최종 모델로 선정하였다

---

## 팀원

- 김진우
- 윤서연
- 오랑거

---

## 프로젝트 목표

- 과거 강우량 데이터를 이용하여 미래 강우량 예측
- 머신러닝과 딥러닝 모델 성능 비교
- 최적의 예측 모델 선정

---

## 사용 데이터

- 여수 2023 데이터
- 여수 2024 데이터
- 여수 2025 데이터
- 통영 2023 데이터
- 통영 2024 데이터
- 통영 2025 데이터

---

## 데이터 전처리

1. 날짜 데이터 변환
2. 결측치 제거
3. 30분 단위 리샘플링
4. 학습 데이터와 테스트 데이터 분리

---

## 비교 모델

### Random Forest

* 여러 개의 결정트리를 이용하는 앙상블 학습 모델
* 비선형 데이터 예측에 강점을 가짐

### LSTM

* 시계열 데이터 분석에 특화된 딥러닝 모델
* 과거 강우량 패턴을 학습하여 미래 강우량 예측

### AutoEncoder

* 데이터의 특징을 압축 및 복원하여 패턴을 학습하는 딥러닝 모델

### Hybrid (1D-CNN + GRU)

* CNN과 GRU를 결합한 하이브리드 모델
* 지역적 특징과 시계열 특징을 동시에 학습

### SVR (Support Vector Regression)

* 서포트 벡터 머신 기반 회귀 모델
* 고차원 데이터에서도 안정적인 예측 수행

### LightGBM

* Gradient Boosting 기반의 머신러닝 모델
* 빠른 학습 속도와 높은 예측 성능 제공

---

## 평가 지표

* MSE (Mean Squared Error)

MSE를 이용하여 실제 강우량과 예측 강우량의 차이를 평가하였다.
